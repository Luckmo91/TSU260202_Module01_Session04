What is CSS

Cascading Style sheets (CSS) is used to format the layout of a webpage


With CSS , you can control the color , font , the size og text , the spacing between elements

Tip: The word cascading means that a style applied to a parent elements will also apply to children
elements within in the parent. So , if you set the color of the body text to "blue" , all headings , paragraphs and
other text elements with the body will also get the same color (unless you specify something else !)
